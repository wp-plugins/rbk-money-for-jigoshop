Adds RBK Money payment gateway to Jigoshop e-commerce plugin.

== Description ==

RBK Money is one of the popular payment gateways in Russia that enables web-shops accepting debit and credit cards, as well as other payment mediums.

This plug-in enables integration of RBK Money with a popular e-commerce solution Jigoshop. The plug-in is 100% free and it was developed when we needed to integrate RBK Money to our website [cardify.ru](http://cardify.ru). If you feel like saying thanks for this plug-in, you are welcome to send any postcard from our website.

== Installation ==

1. Ensure you have latest version of [Jigoshop](http://www.jigoshop.com/) plugin installed
2. Unzip and upload contents of the plugin to your `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to the settings of Jigoshop, find there a tab with payment gateways and fill in your RBK Money merchant ID

== Frequently Asked Questions ==

= How do I get RBK Money merchant ID? =

You should apply for it at their website: http://rbkmoney.ru
